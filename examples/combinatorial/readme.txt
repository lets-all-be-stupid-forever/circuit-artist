This folder shows how to easily create simple tests for (combinatorial)
circuits.

It will read the "ports" and "cases" global variables and check the
output values given the inputs, just as is done in many campaign levels.

There are some examples in the folder that you can read / explore and
modify.

The key script is "_framework.lua" that implements the tests / drawing
routines. Feel free to customize it for better display / customization.
